CSS Specificity

What CSS ???

CSS ย่อมาจาก Cascading Style Sheet คือภาษาที่ใช้เป็นส่วนของการจัดรูปแบบการแสดงผลเอกสาร HTML โดยที่ CSS กำหนดรูปแบบ ของเว็บ ได้แก่ สีของข้อความ สีพื้นหลัง ประเภทตัวอักษร และการจัดวางข้อความ ซึ่งการเขียน css ก็สามารถเขียนได้หลายๆแบบ เช่นการเขียนโดยใช้ selector หรือจะเป็นการเขียนแบบ inline ซึ่งได้ผลลัพธ์ที่ออกมาก็สามารถใช้งานได้เหมือนกัน แต่บางคนอาจจะเคยเจอเวลาที่เรา override style นั้นแล้ว แต่ทำไมไม่ได้ผลลัพธ์ตามที่เราต้องการ คนส่วนใหญ่เลยแก้ปัญหาโดยการเติม !important เข้าไป ซึ่งมันไม่ดีเอาซะเลย เพราะเมื่อ code เรามากขึ้น แล้วเราเอาแต่ใช้ !important จะทำให้ style ทับกันเละยิ่งกว่าเดิม เราจึงจำเป็นที่จะต้องรู้ว่า browser มีมันวิธีการเลือกใช้ style ยังไง เพื่อที่จะลดการทับกันของ style เราได้

CSS Specificity

อะไรคือ CSS Specificity ?

คือค่าบางอย่างที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ ซึ่งมันมีวิธีที่จะคำนวนค่าความสำคัญออกมาเพื่อให้ browser รู้ได้

แล้วจะคำนวณค่ายังไง?

การคำนวณนี้เราจะใช้หลักดูค่าพลัง หรือความสำคัญรวมของแต่ละตัวโดยที่ทุกตัวเริ่มมามีค่าพลัง 0 และแต่ละการเขียน style มีค่าพลังดังนี้

# Style Specificity
1 Inline style 1000
2 ID selectors 100
3 Class selectors, Attributes selectors, Pseudo-classes 10
4 Type selectors, Pseudo-elements 1
5 Universal selector, Combinators, Negation pseudo-clas 0

Example

CODE


                      <div id="page"> 
                        <div class="box">
                          <p> Hello World </p>
                        </div>
                      </div>

CSS Style


                      #page .box {
                        color: green;
                      }
                      #page div {
                        color: red;
                      }
